فرم‌ساز پیشرفته با امکان اتصال به ثبت‌نام وردپرس و مدیریت کامل فرم‌ها

== Description ==

**اولین فرم** یک افزونه قدرتمند برای ساخت فرم‌های نامحدود در وردپرس است که امکان ثبت‌نام خودکار کاربران را فراهم می‌کند.

= امکانات اصلی =

* ساخت فرم‌های نامحدود با نام و توضیحات دلخواه
* افزودن فیلدهای مختلف: متنی، کشویی، انتخابی و آپلود فایل/عکس
* فیلدهای اجباری پیش‌فرض: نام، نام خانوادگی و شماره موبایل
* ثبت‌نام خودکار کاربر هنگام ارسال فرم (قابل تنظیم)
* آپلود فایل با محدودیت ۲ مگابایت
* مدیریت و مشاهده ثبت‌نام‌ها
* خروجی اکسل (CSV) از ثبت‌نام‌ها
* ویرایش اطلاعات ثبت‌نام‌ها
* پشتیبانی از کاربران مهمان
* آپدیت خودکار افزونه

= نحوه استفاده =

1. افزونه را نصب و فعال کنید
2. از منوی "اولین فرم" در پیشخوان وردپرس استفاده کنید
3. فرم جدید بسازید و فیلدهای مورد نیاز را اضافه کنید
4. از شورت‌کد `[business_registration_form id="1"]` برای نمایش فرم استفاده کنید

= شورت‌کد =

`[business_registration_form id="فرم_آی_دی"]`

= پالت رنگی =

* رنگ اصلی: نارنجی #ff3800
* رنگ مشکی: #0c0c0c  
* رنگ سفید: #f2f2f2
* رنگ آبی: #0073ff

== Installation ==

1. فایل افزونه را در پوشه `/wp-content/plugins/first-form/` آپلود کنید
2. افزونه را از طریق منوی 'افزونه‌ها' در وردپرس فعال کنید
3. از منوی "اولین فرم" در پیشخوان استفاده کنید

== Frequently Asked Questions ==

= آیا امکان آپلود فایل وجود دارد؟ =

بله، امکان آپلود فایل و عکس با محدودیت ۲ مگابایت وجود دارد.

= آیا ثبت‌نام خودکار کاربر فعال است؟ =

بله، هنگام ساخت فرم می‌توانید گزینه ثبت‌نام خودکار را فعال کنید.

= آیا خروجی اکسل از ثبت‌نام‌ها ممکن است؟ =

بله، در بخش "ثبت‌نام‌ها" دکمه خروجی اکسل وجود دارد.

= آیا فیلدهای اجباری قابل تغییر هستند؟ =

فیلدهای نام، نام خانوادگی و شماره موبایل به صورت پیش‌فرض در هر فرم وجود دارند.
